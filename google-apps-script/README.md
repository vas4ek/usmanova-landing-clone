# Приём заявок с формы в Google Таблицу

Заявки с лендинга будут падать новой строкой в твою Google-таблицу: **Дата | Имя | Телефон | Цель**.

## Шаги (один раз, ~10 минут)

### 1. Создай таблицу
1. Открой https://sheets.new — создастся пустая Google-таблица.
2. (необязательно) переименуй лист в `Заявки`.

### 2. Добавь скрипт
1. В таблице: меню **Расширения → Apps Script**.
2. Удали весь код из `Code.gs`, вставь содержимое файла `Code.gs` из этой папки.
3. Нажми **Сохранить** (значок дискеты).

### 3. Разверни как веб-приложение
1. Справа вверху **Развернуть → Новое развёртывание**.
2. Шестерёнка → тип **Веб-приложение**.
3. Настройки:
   - **Запуск от имени:** Я (твой аккаунт)
   - **У кого есть доступ:** **Все** (Anyone)
4. **Развернуть** → разреши доступ (Google спросит подтверждение, нажми «Дополнительно → Перейти… → Разрешить»).
5. Скопируй **URL веб-приложения** — он оканчивается на `/exec`.

### 4. Подключи к сайту
Пришли мне этот `/exec` URL — я вставлю его в `index.html` (`const SHEET_URL`) и запушу.
Либо сам: открой `index.html`, найди строку
```js
const SHEET_URL = 'PASTE-YOUR-APPS-SCRIPT-EXEC-URL';
```
вставь туда URL, затем `git add index.html && git commit -m "wire sheet url" && git push`.

### 5. Проверь
Отправь тестовую заявку на сайте → в таблице появится новая строка.

> Примечание: форма шлёт данные «вслепую» (`no-cors`) — это нормально для Apps Script,
> строка всё равно добавляется. Если строки не появляются — открой `/exec` URL в браузере,
> должно написать «USMANOVAFIT lead endpoint is live».
